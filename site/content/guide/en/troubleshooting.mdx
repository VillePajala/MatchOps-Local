This guide helps you resolve common issues with MatchOps.

## Data Issues

### Data Not Saving

**Problem**: Changes aren't persisting between sessions.

**Solutions**:
- Check that the app has storage permissions in Android settings
- Verify you have available storage space on your device
- Try manually saving before closing the app

### Lost Data

**Problem**: Previous games or players have disappeared.

**Causes**:
- App data was cleared in Android settings
- App was uninstalled and reinstalled
- Using a different device than before

**Recovery**:
- Restore from backup if available
- Check if you're on the correct device
- Contact support if data should exist

### Import/Export Not Working

**Problem**: Can't import backup or export data.

**Solutions**:
- Check file format (JSON for backups, xlsx for Excel)
- Ensure file isn't corrupted
- Try a smaller data subset
- Check that the app has file access permissions

## Performance Issues

### App Running Slowly

**Problem**: The app is sluggish or unresponsive.

**Solutions**:
1. Close other apps running in the background
2. Restart the app
3. Clear app cache in Android settings
4. Reduce stored data (archive old seasons)

### Timer Not Accurate

**Problem**: The game timer seems off.

**Causes**:
- Device was sleeping
- App was in background
- Battery saver mode throttled the app

**Solutions**:
- Keep the app in the foreground during games
- Enable wake lock feature in app settings
- Don't switch apps during active timing
- Disable battery optimization for MatchOps

## Display Issues

### Screen Orientation Problems

**Problem**: Layout looks wrong in portrait/landscape.

**Solutions**:
- Lock device orientation to your preference
- Use the app in recommended orientation (landscape for tablet, portrait for phone)
- Restart the app after rotation if display is incorrect

### Elements Missing or Cut Off

**Problem**: Can't see all the interface elements.

**Solutions**:
- Check device display settings (font size, display scaling)
- Try fullscreen mode
- Check device screen size compatibility

## Connectivity Issues

### Offline Mode Not Working

**Problem**: App doesn't work without internet.

**Solutions**:
- Ensure you've opened the app at least once with internet connection
- The app needs to download resources on first launch
- Try reinstalling the app

<Callout type="tip">
After first launch, the app should work offline. Test by enabling airplane mode and reopening.
</Callout>

## Android-Specific Issues

### Background Restrictions

Common issues:
- Timer pauses when app is in background
- Notifications not appearing

**Solutions**:
- Disable battery optimization for MatchOps
- Grant all requested permissions
- Keep app in foreground during games
- Check Android's adaptive battery settings

### Storage Permission Issues

**Problem**: App asks for storage permission repeatedly or can't save files.

**Solutions**:
- Grant storage permissions in Android Settings > Apps > MatchOps
- On Android 13+, grant "Photos and videos" or "Files" permission as needed
- Restart the app after granting permissions

## Getting More Help

### Before Contacting Support

Gather this information:
1. Device model and Android version
2. App version (found in Settings)
3. Steps to reproduce the issue
4. Any error messages
5. Screenshot if possible

### Reset Options

If issues persist:

1. **Soft reset**: Clear app cache in Android settings, reopen
2. **Data export**: Save your data first via backup
3. **Clean reinstall**: Uninstall and reinstall from Play Store
4. **Full reset**: Clear all app data (last resort)

<Callout type="warning">
Always backup your data before attempting any reset procedures.
</Callout>
