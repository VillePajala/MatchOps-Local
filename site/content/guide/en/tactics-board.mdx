The Tactical Board in MatchOps helps you visualize and communicate plays, formations, and strategies to your team.

## Accessing the Tactical Board

- During games for quick tactical discussions
- At halftime to show adjustments
- Before games to set up formations

## Common Uses

### Formation Setup

Use the tactics board to:
- Show starting formation
- Illustrate defensive shape
- Plan attacking positioning

### Set Pieces

Draw set piece routines:
- Corner kicks
- Free kicks
- Throw-ins
- Kickoffs

<Callout type="tip">
Practice your set piece explanations before games so you can quickly draw them during matches.
</Callout>

### Movement Patterns

Illustrate player movement:
- Running paths with arrows
- Passing sequences
- Defensive coverage rotations

## Tactical Discs

### Disc Types

The tactics board provides different disc types for visualization:

- **Home Team Discs** - Your players
- **Opponent Discs** - Opposition players (different color)

### Placing Discs

1. Select the disc type from the toolbar
2. Tap on the field to place
3. Drag to reposition
4. Double-tap a disc to remove it

### Ball Placement

Position the ball on the field:

1. Select the ball tool from the toolbar
2. Tap where you want to place it
3. Drag to move during play explanations
4. Use for set pieces, restart positions, etc.

## Working with Players

### Moving Players

In tactics mode:
1. Tap and hold a player
2. Drag to new position
3. The player snaps to the new location

### Undo/Redo

The Tactical Board has its own separate Undo/Redo history:

- Use undo to reverse recent changes
- Redo to restore undone changes
- Independent from the main field view

## Saving and Sharing

### Screenshot

Tap the camera icon (ðŸ“·) on the field to create a screenshot of the field with game details. This works in both normal field view and tactics mode.

### Clear Canvas

To start fresh:
- Use the **Clear Drawings** button to remove all drawings
- Use **Reset Field** to restore original player positions

## Practical Tips

### During Halftime

Use the tactics board to:
1. Review first-half issues
2. Draw adjustments visually
3. Show new marking assignments
4. Plan second-half changes

### Pre-Game

Before matches:
- Set up formation
- Go through game themes

## Field Types

The tactics board displays different field layouts based on your game type:

- **Soccer field** - Standard pitch with penalty areas, goal boxes, and center circle
- **Futsal court** - Smaller indoor court with arc-shaped penalty areas

The field type is determined by the game's sport type setting (Soccer or Futsal) when you create the game.
