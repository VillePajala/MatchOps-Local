The Tactical Board in MatchOps Local helps you visualize and communicate plays, formations, and strategies to your team.

## Accessing the Tactical Board

Open the Tactics Mode by clicking the **Tactics Mode** button in the control bar. Use it:
- During games for quick tactical discussions
- At halftime to show adjustments
- Before games to set up formations

## Drawing Tools

### Basic Drawing

<Steps>
<Step title="Enter Draw Mode">
Tap the pencil icon to activate drawing mode.
</Step>

<Step title="Choose Color">
Select from available colors for your lines and shapes.
</Step>

<Step title="Draw">
Use your finger or stylus to draw on the field.
</Step>

<Step title="Exit">
Tap the pencil icon again to exit drawing mode.
</Step>
</Steps>

### Available Tools

- **Freehand** - Draw any shape
- **Lines** - Straight lines for passing lanes
- **Arrows** - Show movement direction
- **Shapes** - Circles, rectangles for zones

## Common Uses

### Formation Setup

Use the tactics board to:
- Show starting formation
- Illustrate defensive shape
- Plan attacking positioning

### Set Pieces

Draw set piece routines:
- Corner kicks
- Free kicks
- Throw-ins
- Kickoffs

<Callout type="tip">
Practice your set piece explanations before games so you can quickly draw them during matches.
</Callout>

### Movement Patterns

Illustrate player movement:
- Running paths with arrows
- Passing sequences
- Defensive coverage rotations

## Tactical Discs

### Disc Types

The tactics board provides different disc types for visualization:

- **Home Team Discs** - Your players (numbered)
- **Opponent Discs** - Opposition players (different color)
- **Goalkeeper Discs** - Specialized GK markers

### Placing Discs

1. Select the disc type from the toolbar
2. Tap on the field to place
3. Drag to reposition
4. Long-press to remove

### Ball Placement

Position the ball on the field:

1. Select the ball tool from the toolbar
2. Tap where you want to place it
3. Drag to move during play explanations
4. Use for set pieces, restart positions, etc.

## Working with Players

### Moving Players

In tactics mode:
1. Tap and hold a player
2. Drag to new position
3. The player snaps to the new location

### Player Indicators

Players show:
- Jersey number
- Current position
- Movement trails (if enabled)

### Undo/Redo

The Tactical Board has its own separate Undo/Redo history:

- Use undo to reverse recent changes
- Redo to restore undone changes
- Independent from the main field view

## Saving and Sharing

### Save Drawings

Your drawings can be:
- Saved with the current game
- Exported as images
- Used in post-game reviews

### Clear Canvas

To start fresh:
- Use the eraser tool for partial clearing
- Tap "Clear All" to remove all drawings
- Reset positions to restore original lineup

## Practical Tips

### During Halftime

Use the tactics board to:
1. Review first-half issues
2. Draw adjustments visually
3. Show new marking assignments
4. Plan second-half changes

### Pre-Game

Before matches:
- Set up formation
- Review opponent tendencies
- Walk through set pieces

### Team Meetings

The tactics board works well for:
- Training session planning
- Video review annotations
- Strategy discussions

<Callout type="info">
Players respond better to visual instructions than verbal explanations alone.
</Callout>

## Field Variations

The tactics board supports different field views:

- **Full field** - See entire pitch
- **Half field** - Focus on attacking or defending
- **Futsal court** - Smaller dimensions for futsal

Switch views based on what you're explaining.
