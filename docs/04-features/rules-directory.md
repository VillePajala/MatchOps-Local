## Overview

The Rules Directory provides quick access to official football and futsal rules during games. Links to authoritative rule sources for referee decisions and rule clarifications.

## Key Components

- `RulesDirectoryModal.tsx` - Modal displaying rule links

## Features

- **FIFA Laws of the Game** - Official football rules
- **Futsal Rules** - Indoor futsal regulations (2024/2025 edition)
- **Quick Access** - One-tap access during games
- **External Links** - Opens official sources in new tab

## Rule Sources

| Sport | Source | Link |
|-------|--------|------|
| Football | FIFA Laws of the Game | theifab.com |
| Futsal | FIFA Futsal Laws | fifa.com |

## User Flow

1. Access Rules from field toolbar or settings
2. Select Football or Futsal rules
3. Opens official rule document in browser
4. Return to app when done

## Use Cases

- Clarify offside rules during disputed calls
- Check substitution rules
- Verify handball regulations
- Review futsal-specific rules (4-second rule, etc.)
