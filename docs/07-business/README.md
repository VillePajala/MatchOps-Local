## Strategy Summary

### Current Approach (First Launch)

```
Model:     Quantity limits + One-time purchase ($9.99)
Free:      1 team, 12-15 players, 10-20 games, 1 season
Premium:   Unlimited everything
Timeline:  ~1 day implementation, then ship
Post-launch: ~2 hours/month maintenance
```

### Revenue Expectations

| Scenario | Year 1 | Year 2 |
|----------|--------|--------|
| Minimal effort | $360 | $800 |
| Active engagement | $1,500 | $3,600 |
| Growth focus | $4,800 | $16,000 |

### Indirect Value (Often Larger)

- Career leverage: +$10-30K salary potential
- Teaching/speaking: $5-20K/year
- Consulting premium: +20-50% rates
- Portfolio credibility: Compounds with each app

## Key Decisions

| Decision | Recommendation |
|----------|----------------|
| Pricing model | Quantity limits (not feature gating) |
| Price point | $9.99 one-time |
| Free access | Local promo codes |
| Cloud sync | Skip for now |
| CoachHub vision | Park it, validate first |
