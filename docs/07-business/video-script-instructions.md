# MatchOps Instructional Video Scripts

**Format**: ElevenLabs TTS narration + screen recording
**Target length**: 3-5 minutes
**Tone**: Friendly, coach-to-coach

`[SCREEN: ...]` = what to show on screen during that section.
`(pause)` = brief natural pause in narration (~1s).

---

## English Script

### 1. Intro (15s)

[SCREEN: App logo / splash, then the main field view with players positioned]

Whether you're coaching your first youth team or your tenth season — keeping track of lineups, game time, match events and substitution times during a match can be a handful. MatchOps is built to make that simple.

### 2. Getting Started (20s)

[SCREEN: Welcome screen → tap "Start without an account"]

When you open the app for the first time, you choose how you want to use it. (pause) You can start right away without creating an account — everything stays on your device. Or, if you want to sync across devices, you can set up cloud sync later in settings. Let's jump in.

### 3. Your Roster (30s)

[SCREEN: Open menu → Manage Roster → show player list → add a player → fill name, jersey number → save → scroll through roster]

First, let's set up your players. Open the menu and tap All Players. (pause) Here you can add each player with their name, nickname, and jersey number. You'll see them appear as colored discs in the player bar at the top of the screen when you return to the main view. (pause) You can add as many players as you need — and you can always come back to edit these later.

### 4. Starting a Game (30s)

[SCREEN: Menu → New Game → Game Settings Modal opens → set team name, opponent, date → select players with checkboxes → configure periods → close settings]

To start a match, open the menu and tap Start New Game. (pause) This opens the new game settings where you set your team name, the opponent, and the date. (pause) Scroll down to select which players are available for this game — just check the boxes. (pause) You can also set the number of periods and how long each one lasts. Once you're set, close the panel and you're ready to go.

### 5. The Field (35s)

[SCREEN: Tap a player disc in the player bar → tap a spot on the field to place them → repeat for several players → tap the formation picker → select a formation → show all players snap into position]

Now here's where it gets fun. (pause) Tap a player in the bar at the top to select them, then tap anywhere on the field to place them. It's that simple. (pause) Or, if you want a quick setup, tap the formation picker in the control bar. Choose a formation and all your selected players snap into position instantly. (pause) To easily change player positions, just select a player and tap their new spot to reposition them.

### 6. Running the Game (45s)

[SCREEN: Tap the timer to start it → timer running → tap log a goal (show the goal event appearing) → log an opponent goal → show score updating → tap timer at period end → start new period]

Tap the timer in the bottom bar to start the clock. (pause) When someone scores, press the Log Goal to log the goal. select the player who scored — you can also add an assist. The score updates automatically. (pause) For opponent goals, log those with the opponent + 1 button. (pause) When the preset substitution interval is getting closer, the timer changes color and you know to prepare for the next substitution. Periods end automatically and the next ones starts when you hit play again. Everything is tracked with timestamps so you have a full record of the match.

### 7. Tactics Board (25s)

[SCREEN: Tap tactics button in control bar → draw arrows/lines on the field → add opponent discs (red) → add home discs (purple) → clear drawings]

Need to sketch out a play? Tap the tactics button to open the drawing tools. (pause) Draw directly on the field to plan set pieces or explain movements. You can add opponent markers and extra formation discs too. The drawing os automatically saved for later use or you can clear everything with one tap.

### 8. After the Game — Stats and Assessments (35s)

[SCREEN: Menu → Stats → show stats table with goals/assists/playing time → close → Menu → Assess Players → show slider assessment for a player (drag a few sliders) → show overall rating]

After the final whistle, open Stats from the menu. (pause) You get a full breakdown of the last game — goals, assists and event timeline. (pause) For a deeper look, open Assess Players. Here you can rate each player's performance on ten dimensions like technique, teamwork, awareness, and decision-making. (pause) These assessments build up over time, giving you a real picture of each player's development.

### 9. Seasons, Tournaments & Export (30s)

[SCREEN: Menu → Manage Seasons & Tournaments → show a season with games listed → show creating a tournament → close → Menu → Stats → tap export button → Excel file downloads]

You can organize your games into seasons and tournaments. (pause) Create a season, link your games to it, and track your progress across the whole campaign. Same goes for tournaments. (pause) And when you need to share your data — for a club report or your own records — export everything to Excel with one tap.

### 10. Closing (15s)

[SCREEN: Return to the main field view with a nice lineup displayed, then show app logo]

That's MatchOps — your pitch-side assistant for lineups, game tracking, and player development. (pause) Install it, set up your team, and focus on what matters most: coaching.

## Finnish Script (Suomenkielinen käsikirjoitus)

### 1. Johdanto (15s)

[SCREEN: Sovelluksen logo / aloitusnäkymä, sitten pääkenttänäkymä pelaajien kanssa]

Oli kyseessä sitten ensimmäinen tai kymmenes kautesi valmentajana — kokoonpanojen, peliajan ja tilastojen seuraaminen ottelun aikana voi olla haastavaa. MatchOps tekee siitä helppoa.

### 2. Alkuun pääseminen (20s)

[SCREEN: Tervetuloa-näkymä → napauta "Aloita ilman tiliä"]

Kun avaat sovelluksen ensimmäistä kertaa, valitset käyttötavan. (pause) Voit aloittaa heti ilman tiliä — kaikki tiedot pysyvät laitteellasi. Jos haluat synkronoida tiedot laitteiden välillä, voit ottaa pilvisynkronoinnin käyttöön myöhemmin asetuksista. Aloitetaan.

### 3. Pelaajat (30s)

[SCREEN: Avaa valikko → Hallitse kokoonpanoa → näytä pelaajalista → lisää pelaaja → täytä nimi, pelinumero → tallenna → selaa listaa]

Lisätään ensin pelaajat. Avaa valikko ja napauta Hallitse kokoonpanoa. (pause) Täällä voit lisätä jokaisen pelaajan nimen, lempinimen ja pelinumeron. Pelaajat näkyvät värillisinä kiekkoina yläpalkissa. (pause) Voit lisätä niin monta pelaajaa kuin tarvitset — ja palata muokkaamaan tietoja milloin tahansa.

### 4. Ottelun aloittaminen (30s)

[SCREEN: Valikko → Uusi ottelu → Otteluasetukset avautuu → aseta joukkueen nimi, vastustaja, päivämäärä → valitse pelaajat → määritä jaksot → sulje asetukset]

Aloita ottelu avaamalla valikko ja napauttamalla Uusi ottelu. (pause) Otteluasetuksissa määrität joukkueesi nimen, vastustajan ja päivämäärän. (pause) Valitse pelaajat tälle ottelulle — ruksaa haluamasi pelaajat. (pause) Voit myös asettaa jaksojen määrän ja pituuden. Kun olet valmis, sulje paneeli ja peli voi alkaa.

### 5. Pelikenttä (35s)

[SCREEN: Napauta pelaajaa yläpalkissa → napauta kohtaa kentällä sijoittaaksesi → toista useammalle pelaajalle → napauta muodostelman valintaa → valitse muodostelma → pelaajat asettuvat paikoilleen]

Tässä alkaa varsinainen toiminta. (pause) Napauta pelaajaa yläpalkissa valitaksesi hänet ja napauta sitten kenttää sijoittaaksesi hänet haluamaasi kohtaan. (pause) Tai jos haluat nopean aloituksen, napauta muodostelman valintaa alapalkissa. Valitse muodostelma, niin kaikki pelaajat asettuvat paikoilleen. (pause) Ottelun aikana voit valita pelaajan ja napauttaa uutta paikkaa siirtääksesi hänet.

### 6. Ottelun seuranta (45s)

[SCREEN: Käynnistä ajastin → ajastin käy → valitse pelaaja kentältä → kirjaa maali → kirjaa vastustajan maali → tulos päivittyy → jakso päättyy → aloita uusi jakso]

Käynnistä kello napauttamalla ajastinta alapalkissa. (pause) Kun joku tekee maalin, valitse pelaaja kentältä ja kirjaa maali — voit lisätä myös syöttäjän. Tulos päivittyy automaattisesti. (pause) Vastustajan maalit kirjataan otteluasetuksista tai tapahtumavalikon kautta. (pause) Jakson päättyessä ajastin ilmoittaa siitä. Aloita seuraava jakso kun olet valmis. Kaikki tapahtumat tallennetaan aikaleimoineen, joten sinulla on täydellinen raportti ottelusta.

### 7. Taktiikat (25s)

[SCREEN: Napauta taktiikkapainiketta → piirrä kentälle nuolia/viivoja → lisää vastustajan merkkejä (punaisia) → lisää omia merkkejä (violetteja) → tyhjennä piirrokset]

Haluatko piirtää pelitilannetta? Napauta taktiikkapainiketta piirtotyökalujen avaamiseksi. (pause) Piirrä suoraan kentälle erikoistilanteita tai liikkeitä. Voit lisätä vastustajan merkkejä ja ylimääräisiä muodostelmakiekkoja. Tyhjennä kaikki yhdellä napautuksella kun olet valmis.

### 8. Ottelun jälkeen — Tilastot ja arvioinnit (35s)

[SCREEN: Valikko → Tilastot → näytä tilastotaulukko maaleilla/syötöillä/peliajoilla → sulje → Valikko → Arvioi pelaajia → näytä liukusäätimet → vedä muutamia → näytä kokonaisarvosana]

Loppuvihellyksen jälkeen avaa Tilastot valikosta. (pause) Näet täydellisen erittelyn — maalit, syötöt ja peliaika jokaiselle pelaajalle. (pause) Syvempää analyysiä varten avaa Arvioi pelaajia. Täällä voit arvioida jokaisen pelaajan kymmenellä osa-alueella, kuten tekniikka, yhteistyö, pelinluku ja päätöksenteko. (pause) Arvioinnit kertyvät ajan myötä ja antavat todellisen kuvan pelaajien kehityksestä.

### 9. Kaudet, turnaukset ja vienti (30s)

[SCREEN: Valikko → Hallitse kausia ja turnauksia → näytä kausi peleineen → näytä turnauksen luominen → sulje → Valikko → Tilastot → napauta vienti-painiketta → Excel-tiedosto latautuu]

Voit järjestää ottelusi kausiin ja turnauksiin. (pause) Luo kausi, linkitä ottelut siihen ja seuraa edistymistäsi koko kauden ajan. Sama toimii turnauksille. (pause) Kun haluat jakaa tietoja — esimerkiksi seuraraporttia varten — vie kaikki Exceliin yhdellä napautuksella.

### 10. Lopetus (15s)

[SCREEN: Palaa kenttänäkymään kauniilla kokoonpanolla, sitten sovelluksen logo]

Tämä on MatchOps — kenttälaitasi kokoonpanojen, otteluseurannan ja pelaajakehityksen hallintaan. (pause) Asenna sovellus, lisää joukkueesi ja keskity siihen mikä on tärkeintä: valmentamiseen.

## Production Notes

- **ElevenLabs voice**: Choose a warm, mid-range male or female voice. Avoid overly formal or robotic tones.
- **Pacing**: The `(pause)` markers are ~1 second natural breaks. ElevenLabs handles these well with a period + space or SSML `<break>` tags.
- **Music**: Consider light background music (royalty-free) at ~20% volume under the narration.
- **Screen recording tips**:
  - Use a clean device/emulator with no notifications
  - Pre-populate some realistic player data (Finnish names work well for both versions)
  - Record at 60fps if possible for smooth interactions
  - Keep mouse/touch movements deliberate and slightly slower than natural
  - Each `[SCREEN:]` block maps to one continuous recording clip
- **Total word count**: EN ~620 words (~4 min), FI ~580 words (~4 min)
